##download

版本 v0.0.1

#### 简介
  * 用于下载放入txt文件中的链接文件

#### 使用
  * python download.py


>python环境为2.7(详见"download.py源代码")

#### 注:
  * 在txt文件内必须按照指定格式存放链接(一行一条)。
  * 链接没有进行任何判断，理论上只要是链接都会下载。
